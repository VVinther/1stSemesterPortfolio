CSS

Cascading Style Sheets er det kodesprog man bruger til at style et HTML dokument med.

Ved at bruge selectors kan man definere hvad man gerne vil style. Der er to hoved selectors #id og .class.

#id er lavet til individuelle elementer der er blevet angivet individuelle id’er, imens .class er lavet til grupper af elementer der er blevet givet bestemte klasser.

Man kan også style alle p, div og section elementer man har i sit HTML ved bare at skrive p, div eller section.

Eksempel på en class selector

.indholdWrapper {
    max-width: 1000px;
    margin: auto;
    padding-top: 30px;
    padding-left: 50px;
    padding-right: 50px;
    padding-bottom: 30px;
}
